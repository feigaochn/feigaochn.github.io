Before Start

As FarBox mentioned in its documentations, this is a highly open and customizable system. So the contents of following only reflect my own experience and you probably should know that my experience is only a few days 8-p

Post Files

The suffix of the filename of a post, should be one of .txt .md .markdown .mk.

File Path

File path by default is the category of the post. For example, /site/it/post1.md is in category it; and /site/it/python/post2.md is in both category it and category python.

File Name

Since /path/filename-without-ext is the defaut url of the post, you should come up with a good name and stick to it. Or, you could set the url metadata of posts.

Metadata

Standard Fields

These may be needed by everyone.

Title: Hello World Date: 1949-10-01 Tags: tag1, tag2 Status: public / draft

The following could also be helpful. (Note that, not every templates respect position, and url are relative to /post/)

Url: [default is the path/filename-without-ext] Toc: yes / no Position: 1024

More on Metadata

Generally, there is no bad to provide sufficient metadata when you started a post. They may be supported some day in future. Easily by processing post.metadata.key of post in the template, you can do a lot of exciting things.

For example, in the list view (i.e. / or /index.html), every post will show its first 300 characters. (OK, not quite true, see also "Post List Preview".) Usually this words won't give an idea what the post is talking about. Many other static blogging system solve this by providing a summary or excerpt field for author to briefly describe the content of the post with a few sentences.

Actually, it takes only one line modification in the template to support showing the excerpt of a post in default view. If you provide the Excerpt field in post like these:

Excerpt: A brief introduction of article.

Also it would be great to support another common used key category. But I'm not sure if this can be done by hacking templates, since currently FarBox parses the category info directly from file path.

Mathjax

Inline math with single dollar $n=\beta$: $n=\beta$.

Equation with double dollars sign $$\alpha = \lim_{x \to 0} f(x)$$: $$\alpha=\lim_{x\to0} f(x).$$

Or fancy code with mathjax syntax:

\int_{x=0}^{\infty} \sin(x) d x

You can even use eqnarray. So good, isn't it!

\begin{eqnarray}
    f(0) & = & 0 \\
    f(1) & = & 1 \\
    f(n) & = & f(n-1) + f(n-2) \text{ for $n \ge 2$}
\end{eqnarray}
