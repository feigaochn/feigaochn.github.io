Replace Words in Multiple Files

Date 2014-06-26 Thu Tags sed / find 
find path -type f -iname '*.md' -exec sed -i "" 's/old_word/new_word/g' "{}" \;

The magic -i option let sed do inplace substitution.

The empty string "" between -i and pattern is for backup. This is only needed in BSD systems like Mac OS.
