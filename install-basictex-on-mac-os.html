Install BasicTeX on Mac OS

Published: Tue 01 July 2014
By Fei Gao

In posts.

tags: latex macos

BasicTeX

The TeX Live 2014 has just been released on June 14. The relevant MacTeX-2014 is also available. But to most users, the 2.4GB of full MacTeX package seems unnecessary. So I turn to a smaller distribution named BasicTeX which is only 95MB.

BasicTeX is a subset of TeX Live; it is much smaller than TeX Live and yet supports most standard TeX typesetting.

Since BasicTeX is installed in /usr/local/texlive/2014basic, you can keep the full version of TeX Live too. Later, you could switch the installed versions via "TeX Distribution" in Mac's "System Preferences".

Installation

BasicTeX

You can download BasicTeX from the above web page and install it by double click. But I personally prefer the package manager brew-cask to handle my apps.

brew cask install basictex

Packages

BasicTeX uses tlmgr to manage its packages the same way as TeX Live. After installation, let us update the packages first. Note that, you need administrator password to do these.

sudo tlmgr update --all

Though BasicTeX comes with commonly used packages, you may need to install others required by your documents. Here I install IEEEtran to write IEEE style papers and stmaryrd to use lots of fancy math symbols.

sudo tlmgr install IEEEtran
sudo tlmgr install stmaryrd

If you are not a fan of command line interface, you can use the GUI tool TeX Live Utility.

brew cask install tex-live-utility

latexmk Script

Some times, it is painful to manually compile a latex file with bib through the correct way.

pdflatex file.tex
bibtex file.aux
pdflatex file.tex
pdflatex file.tex

So I usually use this nice script latexmk.

latexmk completely automates the process of generating a LATEX document. Given the source files for a document, latexmk issues the appropriate sequence of commands to generate a .dvi, .ps, .pdf or hardÂ­copy version of the document.

sudo tlmgr install latexmk

# compile
latexmk -pdf file.tex

# clean up all nonessential files, except dvi, ps and pdf files.
latexmk -c
