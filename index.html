Modify Mac OS System Volume in CLI

Published: Fri 08 August 2014
By Fei Gao

In posts.

tags: macosx

When using in-ear headphones, the Mac OS system sound could still be very loud even if we lower the volume down to one quarter by "Option + Shift + VolUp/VolDown". So I googled a solution and write a short script to help control the volume.

#!/bin/bash

# Thanks to http://apple.stackexchange.com/questions/63253/lowest-volume-is-still-too-loud-how-can-i-make-it-even-lower
#
# osascript -e "set Volume $1"
# $1 in range [0.0, 7.0]

if [[ "$#" -eq 0 ]]; then
    osascript -e "get volume settings"
elif [[ "$#" -eq 1 && "$1" -gt 0 && "$1" -lt 100 ]]; then
    osascript -e "set volume $1*7/100"
else
    echo 'Usage:'
    echo '  volume      Get current system volume settings'
    echo '  volume v    Set output volume to v, v in range [0.0, 100.0]'
fi
